# Power BI Dashboard

## Netflix Content Intelligence Platform

This folder contains the final interactive Power BI dashboard developed for the Netflix Content Intelligence Platform project.

The dashboard transforms cleaned and normalized Netflix metadata into an interactive Business Intelligence solution for analyzing content trends, genres, ratings, countries, directors, actors, and release patterns.

---

## Dashboard Features

- Executive KPI Cards
- Movies vs TV Shows Analysis
- Content Release Trend
- Genre Distribution
- Country-wise Content Analysis
- Content Ratings Distribution
- Top Directors
- Top Actors
- Movie Duration Distribution
- Interactive Filters (Type, Release Year, Rating, Country)
- Business Insights Panel

## Data Source

The dashboard uses the cleaned and normalized Netflix dataset stored in MySQL.

Main tables include:

- clean_netflix_titles
- actors
- title_actor
- countries
- title_country
- genres
- title_genre
